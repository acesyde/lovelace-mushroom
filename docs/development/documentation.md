### Installation

Install python

```sh
sudo apt install python3 python3-pip
```

Install mkdocs and plugins

```sh
pip install mkdocs mkdocs-material mkdocs-awesome-pages-plugin
```

Update the path in your `.bashrc`

```sh
PATH="$PATH:/home/{you_linux_user}/.local/bin"
```

### Launch

```sh
mkdocs serve
```
